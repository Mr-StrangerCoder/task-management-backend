A full-stack Task Management Application built with React.js 
on the frontend and Node.js/Express on the backend. Features 
include user authentication with JWT, role-based access control, 
and full CRUD operations for tasks. The MySQL database is hosted 
on Aiven Cloud with SSL encryption and managed via Sequelize ORM. 
Frontend is deployed on Vercel and the backend REST API is hosted 
on Render.
